# taiyo-style スキル選び方ガイド

## まず何をするか

- 既存の文章がある → `/taiyo-analyzer` で分析 → `/taiyo-rewriter` で改稿
- ゼロから作る → 下の表から選ぶ

## 作りたいもの別

| 作りたいもの | スキル |
|-----------|-------|
| ヘッドライン（タイトル） | `/taiyo-style-headline` |
| 箇条書き（メリット・特徴） | `/taiyo-style-bullet` |
| LP 本文 | `/taiyo-style-lp` |
| PS（追伸） | `/taiyo-style-ps` |
| セールスレター全体 | `/taiyo-style-sales-letter` |
| ステップメール | `/taiyo-style-step-mail` |
| VSL 台本（動画セールス） | `/taiyo-style-vsl` |

## 推奨フロー

```
既存素材 → /taiyo-analyzer → /taiyo-rewriter → /taiyo-style-[用途]
```
