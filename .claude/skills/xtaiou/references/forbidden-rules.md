# xtaiou 禁止ルール

## 表現の禁止事項

### 1. 断定・誇大表現 (CRITICAL)

以下の表現は絶対に使用禁止:

| 禁止表現 | 理由 | 代替表現 |
|---------|------|---------|
| 「絶対に」 | 誇大広告 | 「高い確率で」「多くの場合」 |
| 「100%」 | 根拠なし | 「ほぼ確実に」 |
| 「必ず儲かる」 | 景表法違反 | 「〜の可能性がある」 |
| 「誰でも簡単に」 | 誤解を招く | 「正しく実践すれば」 |
| 「今すぐ」(金銭系) | 煽り過度 | 「まずは〜から始める」 |
| 「楽して稼げる」 | 信用毀損 | 「効率的に」 |

### 2. 収益保証 (CRITICAL)

- 「月収XX万円保証」
- 「XX日で元が取れる」
- 「リスクゼロ」
- 「損することはない」

→ 海外事例の「紹介」として書く。読者個人の成果を保証しない。

### 3. 法的リスク表現

| カテゴリ | 禁止例 | 理由 |
|---------|--------|------|
| 薬機法 | 「〜が治る」「〜に効く」 | 医薬品的効能の標榜 |
| 景表法 | 「業界No.1」(根拠なし) | 優良誤認 |
| 特商法 | 価格を隠して誘導 | 不当表示 |
| 金商法 | 「確実に利益」 | 断定的判断の提供 |

### 4. ハッシュタグ

- ツイート文にハッシュタグは一切使用しない
- トミースタイルではハッシュタグ = スパム的印象

### 5. 絵文字

- 記事本文では絵文字を使用しない
- ツイート文でも原則不使用（最大1個まで許容）

### 6. 他者への言及

- 他人の誹謗中傷は厳禁
- 競合サービスの名指し批判は禁止
- 「〜よりも優れている」等の比較表現は避ける

### 7. 政治・宗教・差別

- 政治的立場の表明
- 宗教的な勧誘
- 性別・人種・年齢等による差別表現
- 特定の国・地域への偏見

## フォーマットの禁止事項

### 1. 構成

- 3セクション以下の短い記事（最低5セクション）
- 導入なしでいきなり本題に入る
- CTAなしで終わる

### 2. 文字数

- 500文字未満の記事は自動エラー
- ツイート文が280文字を超える

### 3. 文体

- 全文が敬体（です・ます）のみ
- 全文が常体（だ・である）のみで語りかけゼロ
- 主語が「私」で始まる自分語り記事
