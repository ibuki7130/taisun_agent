## なぜ「ファイルパス参照 + 全文読み込み指示」が最善か

| 方法 | メリット | デメリット |
|------|---------|-----------|
| mdファイルを丸ごと貼り付け | 確実に全文がコンテキストに入る | プロンプトが巨大化（40KB+）、コンテキスト圧迫 |
| ファイルパスだけ指示 | プロンプトが軽い | AIが部分しか読まない可能性あり |
| **ファイルパス + 「全文を分割読みせよ」指示** | **軽量 + 確実** | なし |

→ **方法3を採用**

## 使い方まとめ

1. このファイルの「実行プロンプト」セクション（``` で囲まれた部分）をコピー
2. Claude Code の新しいセッションで貼り付けて実行

4. 「BUILD_TARGETを入力してください」と聞かれるので、作りたいシステムを入力
5. STEP 1 から自動実行開始
