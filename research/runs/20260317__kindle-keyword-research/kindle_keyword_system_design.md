## STEP 2：トレンドKW → Kindleキーワード 変換プロセス

### 重要：直接転用はNG

```
❌ NG: Googleトレンド急上昇「ChatGPT o3」→ そのままKindleキーワードに使う
✅ OK: Googleトレンド急上昇「ChatGPT o3」→ 「なぜ検索されているか」を分析
       → 「AIツール 最新 副業 使い方」としてKindleキーワードに変換
```

**理由:** Google検索意図（情報収集）とAmazon検索意図（購買・解決）は異なる

### 変換フロー

```
トレンドワード
  ↓ ①「なぜ検索されているか」を分解
  ↓ ②「読者の悩み・欲求」に翻訳
  ↓ ③ Amazonサジェストで実際の検索語句を確認
Kindle購買キーワードとして確定
```

### フィルタリング基準

**OK（Kindleに転用可能）**
- ✅ ビジネス・副業・お金に関連している
- ✅ 学習・成長・スキルアップに関連している
- ✅ 悩み・問題解決系（恋愛、人間関係、健康含む）
- ✅ スピリチュアル・運気・占い系
- ✅ 社会問題・時事問題（対処法・解説系）

**NG（Kindleに転用不可）**
- ❌ 芸能人スキャンダル（一過性すぎる）
- ❌ スポーツ試合結果（書籍需要ゼロ）
- ❌ 飲食店・地域情報（書籍化困難）
- ❌ 特定企業の決算情報（ニッチすぎる）


## STEP 3：競合分析

### Amazon検索での競合チェック手順

```
手順:
1. amazon.co.jpの検索バーにキーワードを入力
2. カテゴリを「Kindle本」に絞り込む（重要）
3. 検索結果の件数を確認（画面上部）

判断基準:
  検索結果 5,000件以下 → ブルーオーシャン ✅
  検索結果 1,000件以下 → 超ニッチ（需要確認が必要）
  検索結果 5,000〜10,000件 → 競合適度 △
  検索結果 10,000件超 → レッドオーシャン ⚠️

追加確認:
  上位5冊のレビュー数が50件以下 → 参入余地あり ✅
  上位本の発売日が古い（2年以上前）→ 最新版需要あり ✅
```

## STEP 4：KDP 7キーワード最適化

### 最重要：7枠を「フレーズ」で埋める

```
❌ NG（もったいない使い方）:
  KW1: 副業
  KW2: AI
  KW3: 初心者
  → 3単語しか検索インデックスされない

✅ OK（フル活用）:
  KW1: 副業 初心者 在宅 AI 稼ぎ方
  KW2: 会社員 40代 副業 副収入 自動化
  KW3: AI エージェント 自動化 ChatGPT 月収
  KW4: フリーランス 独立 副業 スキル不要
  KW5: 生成AI 副業 2026年版 最新
  KW6: AIツール 使い方 収益化 完全ガイド
  KW7: FIRE 副業 早期退職 セミリタイア
  → 30〜40キーワードが検索インデックスされる
```

### 7欄の役割分担

| 欄 | 役割 | 内容 |
|----|------|------|
| 1 | メイン複合KW | メインキーワード + 類義語 + ロングテール |
| 2 | ターゲット属性 | 対象者（会社員・40代・主婦等）+ テーマ |
| 3 | 悩み・問題解決 | 読者の悩み・欲求に直結するワード |
| 4 | 関連スキル/手法 | 本の内容に関連する具体的スキル名 |
| 5 | 年版・最新 | 「2026年版」「最新」「完全版」等 |
| 6 | 競合本参照KW | 上位競合本のタイトルワードを引用 |
| 7 | 周辺テーマ | 読者が合わせて検索する関連テーマ |

### KDP規約上の禁止事項

```
❌ タイトルと同じキーワードの繰り返し
❌ 「新刊」「配信中」などの時間関連語
❌ 「ベストセラー」「おすすめ」等の主観的評価
❌ 他の著者名・ブランド名
❌ 誇大広告表現
❌ 引用符・括弧の使用
```

### カテゴリ選択戦略

```
戦略:
  → メインカテゴリ: 実態に合った正確なカテゴリ（規約遵守）
  → サブカテゴリ: より細分化された小カテゴリ（競合が少ない）

目的: 「ベストセラー1位」バッジ獲得
  → ベストセラーバッジ → 検索結果での視認性向上 → 間接的SEO効果

2026年 狙い目カテゴリ例:
  AI系: コンピュータ・IT > AI・機械学習
  副業: ビジネス・経済 > 起業・開業
  投資: ビジネス・経済 > 投資・資産形成
  スピリチュアル: 趣味・実用 > 占い・スピリチュアル
  SNS: ビジネス・経済 > インターネット・SNSマーケティング
```

## STEP 5：タイトル生成フォーミュラ

### Amazon A10アルゴリズムの重み付け

```
1位: タイトル内キーワード（最強）
2位: サブタイトル内キーワード
3位: 7バックエンドキーワード
4位: 説明文内キーワード
5位: カテゴリ（ランキング競合環境を決定）
```

**→ タイトルに最重要KWを配置することが最優先**

### 売れるKindleタイトル公式

```
基本公式:
  [最重要KW（タイトル冒頭）]: [ターゲット] + [解決策] + [具体的数字/期間]
```

### パターン別テンプレート

| パターン | テンプレート | 例 |
|---------|------------|-----|
| 数値・実績型 | [テーマ] 月収[X]万円：[期間]で実現する[属性]のための完全ガイド | 「AIエージェント副業 月収100万円：3ヶ月で実現するサラリーマンのための完全ガイド」 |
| 課題解決型 | [悩み直撃KW]を解決する[テーマ]の[具体的ステップ] | 「新NISAだけじゃ足りない 2026年イラン紛争時代の資産防衛投資術」 |
| トレンド入門型 | [トレンドKW]革命 [内容をひとこと]完全入門 | 「Vibe Coding革命 AIと対話してアプリを作る新しいプログラミング完全入門」 |
| 年版・限定型 | [年] [テーマ]完全版 [サブタイトル] | 「2026年龍神大運期 守護龍占術完全版 愛・仕事・金運の全て」 |
| 属性特化型 | [ターゲット属性]のための[テーマ] [具体的成果] | 「エンパス・ライトワーカーとして生きる HSPが才能に変わる完全入門」 |
| 問題発見型 | なぜ[問題]のか？[解決策] | 「なぜあなたのSNS運用は伸びないのか？AI自動化で解決する実践ガイド」 |

### 効果的な修飾語

| 修飾語 | 効果 | 配置位置 |
|--------|------|---------|
| 「2026年版」「最新版」 | 鮮度・信頼性 | タイトル末尾 or サブタイトル |
| 「完全版」「完全ガイド」 | 網羅性 | サブタイトル |
| 「入門」「初心者向け」 | ターゲット限定 | タイトル or サブタイトル |
| 「〇〇のステップ」「〇〇のルール」 | 具体性・信頼性 | サブタイトル（数字と組み合わせ） |
| 「会社員でも」「主婦でも」 | 共感・ハードル低下 | サブタイトルのリード部分 |
